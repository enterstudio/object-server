Return Value

A newly initialised proxy for the specified type.

Remarks

This overload is for objects whose PrimaryKey is of type Identity or Guid (where ObjectServer handles primary key initialisation). If the primary key type is Defined then this overload will throw an exception.

The actual objects returned from a call to Create are not of the defined type but of a proxy class which sub-classes the persisted object and implements the abstract properties. This is transparent as long as no access to the type occurs. If this is done then it is nessecary to access the proxy's base type to get back to the defined type.
