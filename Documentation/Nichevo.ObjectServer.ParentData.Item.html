Gets or sets the ServerObject that is a Parent object.

public ServerObject this[
   string key
] {get; set;}

Parameters

key
The name of the property in the ServerObject which exposes this Parent.

Property Value

A ServerObject that is a Parent object or if no parent exists null.

Remarks

On object access if a parent exists but has not be retrieved by ObjectServer then it will be retrieved and returned. Otherwise the existing object will be returned. If you wish a null value to be set the assign null to the parent. When an object is assigned as a parent, ObjectServer will locate the correct children collection and automatically append the current object.
