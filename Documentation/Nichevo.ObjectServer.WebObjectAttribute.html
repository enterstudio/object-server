WebObjectAttribute Class

Adding this attribute to a field allows the field to be loaded by ObjectTransaction.SelectWebObjects directly from a HttpRequest

For a list of all members of this type, see WebObjectAttribute Members.

public sealed class WebObjectAttribute : Attribute

Remarks

If applied to private fields then a call to ObjectTransaction.SelectWebObjects will leave the attribute field as untouched. Care must be taken not to use ServerObject's from different transactions using this method.

Example

In the example below a simple web form contains a ServerObject marked with WebObjectAttribute

            public class MyPage : System.Web.UI.Page
            {
            	[WebObject]
            	protected ServerObject myObject;
            }
