Retrives a collection of objects from the data source of the specified type that meet the specified constraint in the given order.

Parameters

type
The type of object to create.
constraint
The constraint to filter the data source by.
ordering
The order of the objects in the collection.

Return Value

A collection of objects of the given type.

Remarks

The objects in the collection are ordered by the given Ordering object.

The actual objects returned from a call to Select are not of the defined type but of a proxy class which sub-classes the persisted object and implements the abstract properties. This is transparent as long as no access to the type occurs. If this is done then it is nessecary to access the proxy's base type to get back to the defined type.
